# Keyboard map

| Context      | Key                              | Action                                           |
| ------------ | -------------------------------- | ------------------------------------------------ |
| Anywhere     | `Ctrl+Shift+V`                   | Open Quick Paste palette                         |
| Anywhere     | (tray left-click)                | Toggle main window                               |
| Palette      | `↑` / `↓`                        | Move selection up / down                         |
| Palette      | `Enter`                          | Copy selection to clipboard + hide palette       |
| Palette      | `Esc`                            | Hide palette                                     |
| Main window  | `Ctrl+K` (planned)               | Focus the search bar                             |
| Snippets     | `Ctrl+S`                         | Save current snippet                             |
| Snippets     | `Ctrl+Enter` (planned)           | Format with Shiki                                |
| Settings     | (click `Save settings`)          | Persist changes                                  |
